# KAM Cockpit — Pilotage Comptes Clés

Cockpit de pilotage des comptes clés (Key Account Management) pour un suivi structuré et partagé avec les KAM et sponsors.

🔗 **[Voir en ligne](https://kam-cockpit.vercel.app)**

## Fonctionnalités

- **Dashboard COMEX** : vision synthétique avec KPIs globaux
- **Onglets par compte** : MAIF, DANONE, PROSOL, KRONENBOURG
- **Calculs automatiques** : moyenne d'avancement, statut 🔴🟠🟢, détection de retards
- **Ultra simple** : les KAM ne saisissent que l'avancement (slider) et la prochaine action
- **Persistance locale** : données sauvegardées dans le navigateur (localStorage)
- **Export/Import JSON** : sauvegarde et partage des données

## Utilisation

Ouvrir `index.html` dans un navigateur ou accéder au déploiement Vercel.

### Pour les KAM
1. Naviguer vers l'onglet de votre compte
2. Glisser le slider d'avancement pour chaque action
3. Renseigner la prochaine action concrète

### Pour le COMEX
Le dashboard se met à jour automatiquement — lisible en 30 secondes.

## Stack technique

- HTML / CSS / JavaScript vanilla
- Zero dépendances
- Déployé sur [Vercel](https://vercel.com)
